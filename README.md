## 📌 Objetivo

Divulgar o trabalho profissional de **Lash Designer**, apresentando informações como:
- Quem é Larissa
- Serviços oferecidos
- Depoimentos de clientes
- Localização
- Formas de contato (WhatsApp)

2. Abra o arquivo `index.html` diretamente no navegador
